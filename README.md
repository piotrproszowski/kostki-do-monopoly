# Kostki do Monopoly

Prosta aplikacja webowa z dwoma kostkami do gry w Monopoly z realistyczną animacją 3D.

## Funkcjonalności

- 🎲 Dwie kostki 3D z realistycznymi kropkami
- ✨ Płynna animacja rzutu kostkami
- 📱 Responsywny design (działa na telefonach i tabletach)
- 🎮 Wiele sposobów rzucania:
  - Kliknięcie przycisku "Rzuć kostkami!"
  - Naciśnięcie spacji lub Enter
  - Kliknięcie/dotknięcie kostki
  - Podwójne kliknięcie kostki
- 📊 Wyświetlanie wyników każdej kostki i sumy

## Technologie

- HTML5
- CSS3 (animacje 3D, gradients, flexbox)
- Vanilla JavaScript
- Responsive design

## Jak używać

1. Otwórz `index.html` w przeglądarce
2. Kliknij "Rzuć kostkami!" lub użyj klawiatury (spacja/Enter)
3. Kostki będą się obracać przez 0.8 sekundy i pokażą wynik

## Struktura plików

```
kostka-do-gry/
├── index.html      # Główna strona HTML
├── style.css       # Stylowanie i animacje CSS
├── script.js       # Logika JavaScript
└── README.md       # Ten plik
```

## Demo na żywo

Możesz przetestować aplikację bezpośrednio w GitHub Codespace lub GitHub Pages.

## Instalacja lokalna

1. Sklonuj repozytorium
2. Otwórz `index.html` w przeglądarce
3. Lub uruchom lokalny serwer:
   ```bash
   python3 -m http.server 8000
   ```
